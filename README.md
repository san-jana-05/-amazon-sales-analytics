# Amazon India Sales Dashboard

## Project Overview
Analysis of Amazon India sales data (Apr - Jun 2022) 
with 1,09,487 orders across multiple product categories.

## Key Insights
- Top Category: Set (highest revenue)
- Top State: Maharashtra
- Top Month: April 2022
- Top Size: Medium (M)
- Total Revenue: ₹7,85,92,678

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Creation
